# Treido Marketplace

Marketplace built with Next.js (App Router), Supabase, Stripe, Tailwind v4, shadcn/ui, and next-intl.

## Quickstart

```bash
pnpm install
pnpm dev
```

## Common commands

```bash
pnpm -s lint
pnpm -s typecheck
pnpm -s styles:gate
pnpm -s test:unit
pnpm -s test:e2e:smoke
```
