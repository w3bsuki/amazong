## Quick Commands

```bash
# Start the main app
pnpm dev          # localhost:3000

# Type check
pnpm -s exec tsc -p tsconfig.json --noEmit

# Run tests
pnpm test:unit                                    # Unit tests with Vitest
REUSE_EXISTING_SERVER=true pnpm test:e2e:smoke    # E2E smoke (reuse dev server)
pnpm test:e2e                                     # Full E2E suite

# This docs site
pnpm docs        # localhost:3002
```
