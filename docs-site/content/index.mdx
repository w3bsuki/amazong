## Quick Commands

```bash
# Start the main app
pnpm dev          # localhost:3000

# Run tests
pnpm test:unit    # Unit tests with Vitest
pnpm test:e2e     # E2E tests with Playwright

# Type check
pnpm -s exec tsc -p tsconfig.json --noEmit

# This docs site
cd docs-site && pnpm dev  # localhost:3002
```
