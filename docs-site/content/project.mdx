## Product Snapshot

Treido is a mobile-first marketplace built with Next.js, Supabase, Stripe, Tailwind CSS v4, and shadcn/ui.

## Current Phase

Production hardening and launch-readiness execution across reliability, UX polish, and operational readiness.

## P0 Flows

- Auth and account lifecycle
- Browse/search/category discovery
- Listing creation and management
- Cart, checkout, and order lifecycle
- Messaging and trust/safety core actions
- Payout eligibility and Stripe Connect onboarding

## Out of Scope

- Native mobile apps
- Auctions/bidding
- Large post-launch platform initiatives

## Done Means

A change is done when behavior is correct, scoped, and verified with the required gates in `docs/QA.md`.
