## Objective

Enable safe, on-platform payments with escrow-style release and seller payouts.

## Key requirements

- Stripe payment intents / checkout
- Webhook-driven order state
- Hold funds until delivery confirmation / dispute resolution

- Refund handling and chargeback playbook
