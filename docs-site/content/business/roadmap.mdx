# Roadmap (Treido.eu)

This roadmap is phase-based:

- **V1:** classifieds-first + boosts (no on-platform payments)
- **V2:** card payments + seller payouts (Stripe Connect)

In scope:

- Commissions on sales

## V2: payments + payouts

In scope:

- Card payments on-platform
- Stripe Connect Express payouts
- Order lifecycle + basic refunds

Hard gates:

- Stripe Connect onboarding verified in production
- Refunds + webhooks verified
- Legal/policy coverage for payments + refunds
