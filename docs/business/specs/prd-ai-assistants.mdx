# PRD: AI Assistants (Listing + Search)

## Objective

Use AI to improve:

- listing quality (better titles, categories, descriptions)
- discovery (semantic search and shopping assistant)

## V1.1 scope (recommended)

- AI Listing Assistant (seller tool):
  - suggested title + attributes
  - category suggestions
  - description rewrite (tone and clarity)

- AI Search Assistant (buyer tool):
  - natural language search
  - summarised top picks
  - optional paid feature / plan perk
