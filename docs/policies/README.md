### Before Launch

1. Replace all `[PLACEHOLDER]` values with actual company information:
   - Company name and registration (EIK)
   - VAT number
   - Registered address
   - Contact phone numbers

2. Legal review recommended (especially for BG versions to ensure compliance with Bulgarian law)

3. Implement on website:
   - `/terms` → Terms of Service
   - `/privacy` → Privacy Policy
   - `/cookies` → Cookie Policy
   - `/seller-agreement` → Seller Agreement
   - `/prohibited-items` → Prohibited Items

4. Add consent checkboxes:
   - Registration: "I agree to Terms and Privacy Policy"
   - Marketing: "I want to receive promotional emails"

### Translation Notes

- Bulgarian versions are the authoritative legal documents for Bulgarian users
- English versions serve international users and as reference
- In case of conflict, Bulgarian law and Bulgarian version prevail for Bulgarian users
