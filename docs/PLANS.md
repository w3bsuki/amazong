# üîß STRIPE SETUP GUIDE

## Step 1: Access Stripe Dashboard

1. Go to [dashboard.stripe.com](https://dashboard.stripe.com)
2. Make sure you're in **Live mode** (toggle in top-left, not "Test")
3. Navigate to **Products** in left sidebar

---

## Step 2: Create Products

You need to create **3 products** (Free tiers don't need Stripe products):

### Product 1: Treido Premium

1. Click **"+ Add product"**
2. Fill in:
   - **Name:** `Treido Premium`
   - **Description:** `For regular sellers. 100 listings, 5% commission, social links, basic analytics.`
   - **Image:** Upload your Premium badge/icon (optional)

3. Add **TWO prices:**

   **Price A - Monthly:**
   - Click "Add another price"
   - **Pricing model:** Recurring
   - **Amount:** `9.99 EUR`
   - **Billing period:** Monthly
   - **Price description:** `Monthly subscription`
   
   **Price B - Yearly:**
   - Click "Add another price"  
   - **Pricing model:** Recurring
   - **Amount:** `99.00 EUR`
   - **Billing period:** Yearly
   - **Price description:** `Yearly subscription (save ‚Ç¨20)`

4. Click **"Save product"**

5. **Copy the Price IDs** (they look like `price_1ABC123...`):
   - Monthly: `price_xxxxxxxxxxxxxxxx`
   - Yearly: `price_yyyyyyyyyyyyyyyy`

---

### Product 2: Treido Pro

1. Click **"+ Add product"**
2. Fill in:
   - **Name:** `Treido Pro`
   - **Description:** `For power sellers. Unlimited listings, 3% commission, full analytics, priority support.`

3. Add **TWO prices:**

   **Monthly:** `29.99 EUR` / month
   **Yearly:** `299.00 EUR` / year (save ‚Ç¨60)

4. **Save** and copy Price IDs

---

### Product 3: Treido Business Pro

1. Click **"+ Add product"**
2. Fill in:
   - **Name:** `Treido Business Pro`
   - **Description:** `For established businesses. Unlimited listings, 2% commission, full dashboard, dedicated support.`

3. Add **TWO prices:**

   **Monthly:** `49.99 EUR` / month
   **Yearly:** `499.00 EUR` / year (save ‚Ç¨100)

4. **Save** and copy Price IDs

---

## Step 3: Configure Webhook

1. Go to **Developers ‚Üí Webhooks** in Stripe
2. Click **"+ Add endpoint"**
3. Fill in:
   - **Endpoint URL:** `https://treido.eu/api/subscriptions/webhook`
   - **Description:** `Treido subscription events`
   
4. Select events to listen to:
   ```
   ‚úÖ checkout.session.completed
   ‚úÖ customer.subscription.created
   ‚úÖ customer.subscription.updated
   ‚úÖ customer.subscription.deleted
   ‚úÖ invoice.payment_succeeded
   ‚úÖ invoice.payment_failed
   ‚úÖ customer.subscription.trial_will_end (if using trials)
   ```

5. Click **"Add endpoint"**

6. **Copy the Signing secret** (starts with `whsec_...`)

---

## Step 4: Set Environment Variables

In Vercel (or your hosting), set these environment variables:

```env
# Stripe Keys (from Stripe Dashboard ‚Üí Developers ‚Üí API keys)
STRIPE_SECRET_KEY=<your-stripe-secret-key>
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=<your-stripe-publishable-key>

# Webhook Secret (from Step 3)
STRIPE_WEBHOOK_SECRET=<your-webhook-secret>

# App URL
NEXT_PUBLIC_APP_URL=https://treido.eu
```
