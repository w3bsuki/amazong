# Moderation Playbook (V1)

## Goals

- Keep the marketplace safe
- Remove prohibited/counterfeit listings quickly
- Reduce scam attempts and harassment

## Report categories

- prohibited items
- counterfeit / replicas
- scam / fraud attempt
- harassment / hate speech
- spam / duplicate listings

## Actions (ladder)

1. Warn
2. Remove listing
3. Temporary restriction (posting or messaging)
4. Suspension
5. Ban (with documentation)

## Counterfeit handling (high priority)

- remove listing quickly
- preserve evidence (listing + photos + chat + order)
- restrict seller pending review

## Appeals (baseline)

- allow a single appeal with evidence
- document final outcome
