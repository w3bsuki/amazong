# Сигурност на профила (Помощ)

> Как да пазите профила си в Treido.

---

## basics: Основи

- Използвайте силна и уникална парола.
- Не споделяйте данните си за вход.
- Пазете имейла си (достъпът до имейл може да се използва за ресет на парола).

## suspicious-activity: Подозрителна активност

Ако подозирате неоторизиран достъп:

1) Сменете паролата си веднага.
2) Проверете последната активност (поръчки, съобщения, обяви).
3) Свържете се с поддръжката възможно най‑скоро.
